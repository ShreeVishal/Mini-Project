These are the methods in executing the application:

1) Extract the content of the zip file into a folder (store it in any location of your preference)
2) Open the folder as a new project in AndroidStudio
3) Either use an emulator to execute the application or connect your mobile (android) device via USB to your machine and run the application on your device directly.
4) Usage of emulator/device can be managed via device manager in AndroidStudio
